# 口袋战场 Pocket Monsters Battle

一款基于德州扑克规则的多人神奇宝贝对战游戏。

## ✨ 核心特性

- 🎮 **多人实时对战** - 支持 2-8 人同时游戏，基于 WebSocket 实时同步
- 🤖 **智能 AI 机器人** - 内置增强型 AI 对手，支持动态加入/移除
- 🎬 **锦标赛动画** - 精美的对战动画系统，逐轮展示玩家对决
- 🌍 **双语支持** - 中文/英文界面切换
- 🎵 **音效系统** - 背景音乐与战斗音效，可调节音量
- 👀 **观战模式** - 筹码耗尽后自动进入观战，继续观看对局
- 📈 **盲注升级** - 每 5 轮自动提升盲注，增加游戏节奏

## 🎯 玩法

- 每位玩家持有 2 张神奇宝贝手牌，加上 5 张公共战场牌，从中选出最优的 5 张组合牌型
- 牌型越强、点数越高，战斗力越大
- 盲注每 5 轮自动提升（10/20 → 20/40 → ... → 500/1000），筹码清零的玩家成为观战者
- 最后存活的玩家获得最终胜利

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动服务器
npm start
# 或使用开发模式（自动重启）
npm run dev

# 浏览器打开
open http://localhost:3000
```

## 🛠️ 技术栈

- **后端**: Node.js + Express + Socket.io 4.x
- **前端**: 原生 HTML / CSS / JavaScript
- **实时通信**: WebSocket (Socket.io)
- **字体**: Press Start 2P (像素风格) + Noto Sans SC (中文)
- **音频**: Web Audio API

## 🎮 游戏功能

### 房间系统
- 创建/加入房间（5 位房间码）
- 房主控制游戏开始
- 实时房间列表显示

### AI 机器人
- 8 个预设角色：小智、小霞、小刚、大木博士、火箭队喵喵、小次郎、小兰、阿桂
- 增强型 AI 策略：
  - 基于手牌强度的动态决策
  - 支持诈唬、加注、全押等策略
  - 防止无限加注循环（每轮最多 3-4 次加注）
  - 模拟思考时间（0.9-2.3 秒）

### 锦标赛动画
- Showdown 阶段自动播放对战动画
- 逐轮展示玩家对决（A vs B，胜者 vs C...）
- 显示双方手牌、最佳牌型、战斗力
- 每场对决 8 秒，间隔 1 秒

### 观战模式
- 筹码耗尽自动进入观战
- 观战者可继续观看游戏进程
- 显示观战提示与状态

### 盲注系统
- 7 个盲注等级：10/20 → 20/40 → 40/80 → 80/160 → 150/300 → 250/500 → 500/1000
- 每 5 轮自动升级
- 实时显示当前盲注与局数

## 🌐 国际化

支持中文/英文切换，覆盖：
- 所有 UI 文本
- 牌型名称
- 游戏阶段标签
- 提示信息

## 📝 开发说明

- 使用 `nodemon` 进行开发（`npm run dev`）
- 所有游戏逻辑在 `shared/gameLogic.js` 中共享
- 客户端使用 Socket.io 事件驱动架构
- 服务端维护房间状态，处理所有游戏逻辑
