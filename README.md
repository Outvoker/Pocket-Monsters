# 口袋战场 Pocket Monsters Battle

一款基于德州扑克规则的多人神奇宝贝对战游戏。

## 玩法

- 每位玩家持有 2 张神奇宝贝手牌，加上 5 张公共战场牌，从中选出最优的 5 张组合牌型
- 牌型越强、点数越高，战斗力越大
- 盲注每 5 轮自动提升，筹码清零的玩家成为观战者
- 最后存活的玩家获得最终胜利

## 牌型（从低到高）

| 牌型 | 说明 |
|------|------|
| 最强先锋 | 单张最大 |
| 同伴出击 | 一对 |
| 双打联盟 | 两对 |
| 三强联合 | 三条 |
| 进化之路 | 顺子 |
| 系出同源 | 同花 |
| 精锐战队 | 葫芦 |
| 四族合璧 | 四条 |
| 同系连击 | 同花顺 |
| 皇家战队 | 同花大顺 |

## 快速开始

```bash
# 安装依赖
npm install

# 启动服务器
node server.js

# 浏览器打开
open http://localhost:3000
```

## 技术栈

- **后端**: Node.js + Express + Socket.io
- **前端**: 原生 HTML / CSS / JavaScript
- **实时通信**: WebSocket (Socket.io 4.x)
