## Why

当前游戏已有基础的卡牌入场动画、属性音效和对战锦标赛动画，但整体视觉反馈仍较单薄，缺少宝可梦系列标志性的战斗特效（如属性技能释放、暴击震屏、进化光效等），导致游戏的沉浸感和刺激感不足。增强特效系统将显著提升游戏的视觉冲击力和可玩性，让每一次操作都充满宝可梦对战的仪式感。

## What Changes

- **属性技能特效系统**: 每种属性（火/水/草/电）在关键操作（raise/allin/获胜）时释放对应的全屏技能动画，如喷火龙的火焰漩涡、水箭龟的水炮、妙蛙花的飞叶快刀、皮卡丘的十万伏特
- **暴击震屏效果**: All-in 时触发屏幕震动 + 闪白效果，模拟宝可梦对战中的暴击命中感
- **公共牌揭示增强**: 翻牌/转牌/河牌阶段切换时增加戏剧性的阶段转场动画，包括精灵球旋转展开效果
- **筹码变化动画**: 赢得筹码时显示金币飞入动画，失去筹码时显示碎裂消散效果
- **连胜/高光时刻**: 连续获胜时触发渐强的光环特效，达成高牌型（四天王/传说阵容等）时播放专属稀有特效
- **待机呼吸动画**: 当前回合玩家的卡牌增加脉冲呼吸光效，明确指示轮到谁行动
- **对战结算增强**: showdown 阶段的胜者展示增加光柱升天 + 粒子爆发效果，失败者增加灰化消散动画

## Capabilities

### New Capabilities
- `type-skill-effects`: 属性技能全屏特效系统 — 火焰漩涡/水炮/飞叶快刀/十万伏特等属性专属动画
- `screen-shake-flash`: 暴击震屏与闪白系统 — All-in 和关键时刻的屏幕震动、闪光反馈
- `phase-transition-effects`: 阶段转场特效 — 翻牌/转牌/河牌的戏剧性切换动画与精灵球展开效果
- `chip-animations`: 筹码变化动画 — 金币飞入和碎裂消散的视觉反馈
- `streak-highlight-effects`: 连胜光环与稀有牌型高光特效
- `active-turn-glow`: 当前回合呼吸脉冲光效指示系统
- `showdown-vfx-enhance`: 对战结算增强特效 — 胜者光柱粒子爆发与败者灰化消散

## Impact

- **前端文件**: `public/css/style.css`（新增大量 keyframe 动画和特效样式）、`public/js/client.js`（特效触发逻辑、粒子系统、震屏实现）、`public/index.html`（新增特效容器 DOM 元素）
- **性能考量**: 所有特效使用 CSS transform/opacity 动画 + requestAnimationFrame 确保 GPU 加速，避免 layout thrashing；粒子数量需设上限防止低端设备卡顿
- **音效集成**: 新特效需与现有 Web Audio API 音效系统配合，属性技能动画同步播放对应音效
- **无服务端改动**: 所有特效均为纯前端实现，不需要修改 server.js 或通信协议
